# 🛡️ PhishVille – Cybersecurity Awareness Simulation

PhishVille is a **scenario-based simulation project** designed to increase users' awareness of cybersecurity threats.  
It consists of a **frontend (React)** and a **backend (Node.js + Express + MongoDB)**.  

---

## 🚀 Features
- Desktop-like user interface
- Simulated applications: email, file downloads, antivirus, browser, and more
- Realistic threat scenarios such as ransomware and phishing
- Task & scoring system
- User authentication and password reset flow (JWT + Nodemailer)

---

## 🛠️ Technologies Used
- **Frontend:** React, Context API, CSS Modules
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Others:** Nodemailer, JWT, Helmet, CORS, Rate Limiter
